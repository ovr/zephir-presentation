Zephir

Zephir - Ze(nd Engine) Ph(p) I(nt)r(mediate)

Zephir — это высокоуровневый язык программирования с открытым исходным кодом для быстрого и простого создания PHP расширений.

Самые видимые различия после PHP

  • Ура больше нет
  • Прощай $
  • Переменная должна быть обьявлена перед использованием

Особенности

  • dynamic/static typing
  • Memory safety
  • Ahead-of-time (AOT) compiler to provide predictable performance
  • Restricted procedural programming, promoting OOP
  • Reduced execution overhead compared with full compilation
